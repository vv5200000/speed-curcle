## 1. 设计理念

### 1.1 核心理念

**"基础扎实，层层叠加"** — 每个模块独立完整，模块之间松耦合，模式之间可切换。

- **核心模块（Core）**：任何模式都必须的基础逻辑，单独存在就是一个可玩的竞速游戏
- **可选模块（Optional）**：在核心基础上增加竞速深度，不影响游戏基础循环
- **PVP模块（Combat）**：独立的对抗系统，可按需加载，不破坏竞速节奏

## 4. 可选模块（Optional）

### 4.1 热量系统（Heat System）⭐

**核心机制**：过弯超速会产生"热量"，热量溢出导致"爆缸"。

#### 热量数据结构

```typescript
interface HeatPool {
  cards: Card[];       // 当前热量池中的卡牌
  maxCapacity: number;  // 容量上限（默认3张）
  isOverheat: boolean;  // 是否爆缸
}

// 热力卡（Heat Card）
interface HeatCard extends Card {
  type: 'heat';
  heatValue: number;   // 产生的热量值（1-3）
}
```

#### 热力卡获取规则

| 触发条件 | 获取数量 |
|----------|----------|
| 过弯速度超过限速1点 | +1张热力卡 |
| 过弯速度超过限速2点 | +2张热力卡 |
| 强制跳档（换档阶段） | +1张热力卡 |
| 使用极限压弯 | +1张热力卡（盲抽） |

#### 热量结算时机

每回合**移动阶段完成后**统一结算弯道热量：

```
过弯速度 = 本回合打出的所有移动卡点数之和
弯道限速 = 弯道格子的speedLimit值

IF 过弯速度 > 弯道限速:
  热力惩罚 = 过弯速度 - 弯道限速
  从玩家手牌中移除热力惩罚张卡牌 → 加入热量池
  IF 热量池已满:
    触发爆缸（见CRASH模块）
```

#### 冷却机制

热量卡**不能直接使用**，可通过以下方式移回热量池：

| 冷却条件 | 冷却数量 |
|----------|----------|
| 处于1档时结束回合 | 全部热力卡冷却 |
| 经过/停在维修站 | 全部热力卡冷却 |
| 打出特定"冷却"卡 | 1张热力卡冷却 |

> **设计意图**：热量既是惩罚也是资源——你可以通过高压操作获得优势，但必须承担热量管理的风险。

### 4.2 换档系统（Gear System）⭐⭐

**核心机制**：档位决定本回合必须打出的卡牌数量。

#### 档位定义

| 档位 | 必须出牌数 | 热量惩罚倍率 | 说明 |
|------|-----------|-------------|------|
| 1档 | 1张 | ×0（最低档，安全） | 最低速度，可冷却 |
| 2档 | 2张 | ×1 | 入门档位 |
| 3档 | 3张 | ×1.5 | 进攻档位 |
| 4档 | 4张 | ×2 | 高风险档位 |
| 5档 | 5张 | ×2.5 | 极限档位 |
| 6档 | 6张 | ×3 | 爆死档位 |

> **热量惩罚倍率**：过弯超速时，热力卡消耗数量 = 超出限速值 × 档位惩罚倍率

#### 换档规则

```
① 平地换档（上下1档）：免费
② 强制跳档（跨2档以上）：
   - 需支付1张热力卡（无热力卡则强制降至1档）
③ 每回合开始时可换档
④ 爆缸后强制重置为1档
```

#### 换牌阶段流程（GEAR模式）

```
出牌阶段:
  requiredCards = 当前档位
  
  IF 手牌数 < requiredCards:
    仍可出牌（本回合欠着）
    下回合强制从1档起步（需支付热力卡才能换档）
  
  ELSE:
    选择 requiredCards 张手牌打出
    结算移动力总和
```

### 4.3 冷胎机制（Tire System）

**核心机制**：比赛开始/出站后，轮胎未达工作温度，过弯限速临时降低。

#### 触发条件

| 触发场景 | 限速影响 |
|----------|----------|
| 第1圈的所有弯道 | 限速 -1 |
| 维修站出来后的第1个弯道 | 限速 -1 |

#### 退出条件

经过**任意一段直道（含氮气区）**后，轮胎视为"达到工作温度"，恢复正常限速。

```
IF 当前圈 == 1 && 当前格 == 弯道:
  speedLimit -= 1

ELIF 刚从维修站出来 && 当前格 == 弯道:
  speedLimit -= 1

ELIF 经过直道格后:
  tireTemp = 'warm'  // 恢复正常
```

### 4.4 尾流加速（Slipstream System）

**核心机制**：紧跟前车可以获得额外移动，且这次额外移动不计入弯道限速。

#### 触发条件

```
IF 移动完成后:
  ① 停在另一辆车的正后方（同车道紧邻）
  ② 或停在相邻前车的相邻格子
THEN:
  触发尾流 → 获得 +2 格额外移动
  尾流移动单独结算（不触发弯道惩罚）
```

#### 尾流移动规则

```
尾流额外移动:
  - 单独向前移动2格
  - 不触发任何格子效果（安全通过）
  - 不计入弯道限速计算
  - 移动完成后可再次触发尾流（链式）
```

---

### 4.5 重心标记（Body Weight System）

**核心机制**：每圈3个标记，可微调移动力±1。

```
玩家每圈拥有3个重心标记

出牌结算后，可消耗1个标记：
  - 将最终移动力 +1（吃掉尾流/追上对手）
  - 将最终移动力 -1（避免超速/精准停靠）

未使用的标记不跨圈保留
```

---

### 4.6 真实摔车（Crash System）

**核心机制**：热量池满时爆缸，受到更严厉惩罚。

#### 爆缸判定

```
IF 热量池已满 && 本回合产生新热量:
  触发爆缸（Engine Blowout）
```

#### 爆缸惩罚（叠加于基础热量惩罚）

| 惩罚项 | 效果 |
|--------|------|
| 档位重置 | 强制降至1档 |
| 移动力 | 本回合移动力归零（停止） |
| 热力卡 | 热量池全部清空 |
| 下回合换档 | 需额外支付1张热力卡才能从1档起步 |

```
// 示例
当前热量池=3/3上限，本回合过弯超速+1：
  → 需要1张新热力卡
  → 热量池爆满 → 触发爆缸
  → 惩罚：档位重置1档 + 热量池清空 + 本回合停止
```

### 4.7 极限压弯（Leaning System）

**核心机制**：主动提高弯道限速，代价是强制盲抽热力卡。

```
过弯前可宣告"极限压弯"：
  弯道限速 +2（更不易超速）
  
  代价：
  - 必须从牌库顶盲抽1张"压力卡"并立刻结算
  - 压力卡点数直接加入本回合移动力（可能超速）
```

#### 压力卡（Stress Card）

| 牌名 | 类型 | 效果 |
|------|------|------|
| 压力卡 | 特殊 | 盲抽1张基础移动卡，直接加入移动力 |

---

### 4.8 钻车缝（Slingshot System）

**核心机制**：在尾流基础上完成超车，可获得热量卡奖励。

```
触发条件（同时满足）：
  ① 本回合触发了尾流
  ② 尾流移动后超越了至少2辆赛车

奖励：
  可将1张热力卡从热量池移回手牌（或直接冷却）
```

## 5. PVP模块（Combat）

> 完全独立的对抗系统，与可选模块无依赖关系。

### 5.1 攻击牌组（Attack Cards）

| 牌名 | 数量 | 效果 | 射程 |
|------|------|------|------|
| 路障投掷 | 4 | 目标下回合-2步 | 前方1-3格 |
| 电磁脉冲 | 3 | 目标跳过下回合出牌阶段 | 全图 |
| 漏油陷阱 | 3 | 指定格放漏油，经过者-1步 | 前方1-4格 |
| 碰撞冲击 | 3 | 紧邻前方玩家后退1格 | 前方1格 |

### 5.2 防御牌组（Defense Cards）

| 牌名 | 数量 | 效果 | 使用时机 |
|------|------|------|----------|
| 装甲护盾 | 4 | 抵消下一次攻击牌 | 响应式 |
| 紧急规避 | 3 | 减少攻击效果1步惩罚 | 响应式 |
| 黑客反制 | 3 | 将攻击反弹给施加者 | 响应式 |

### 5.3 响应窗口（Reactive Defense）

```
① 攻击方打出攻击牌，指定目标
② 系统广播"防御响应窗口"（5秒倒计时）
③ 目标玩家可选择：
   - 打出防御牌（响应）
   - 跳过（不响应）
④ 结算防御结果：
   - 护盾 → 完全抵消
   - 紧急规避 → 减少1步惩罚
   - 黑客反制 → 反弹给攻击方
⑤ 若无响应，攻击效果生效
```

### 5.4 负面效果（Debuffs）

| 效果名 | 触发条件 | 持续时间 | 效果 |
|--------|----------|----------|------|
| 减速 | 路障投掷命中 | 1回合 | 掷骰/出牌-2步 |
| 沉默 | 电磁脉冲命中 | 1回合 | 跳过出牌阶段 |
| 漏油 | 踩中漏油格 | 立即 | -1步 |
| 碰撞 | 被碰撞冲击命中 | 立即 | 后退1格 |

| `turn:end` | S→C | `{ nextPlayerId }` | 回合结束 |

| `heat:gained` | S→C | `{ playerId, amount, reason }` | 热量增加 |
